II. ğŸ›‘ Mistake 1: Skipping the **Emergency Fund**

Idhu dhaan **number one mistake** da. **SIP** start panradhu **good**, aana **Emergency Fund** (6 months expense) illama start panninaa, **periya risk**-a edukka ready aaguranga nu artham.

The Cost of Selling Early

**Urgent money** venum-na, nee **SIP**-la irukkura **units**-a **loss**-la kooda **sell** panna vendi varum. Adhu unga **long-term growth**-a **destroy** pannidum. **Emergency Fund**-a first **complete** pannunga.

III. ğŸ›‘ Mistake 2: Chasing **Past Returns**

"Andha fund **last year 50%** return kuduththuchu, adhula podalaam!" nu yosikradhu **periya thappu**.

Past Performance is NOT Future Guarantee

**Past Returns** **Future Returns** illai. **Risk**-a paaththu dhaan invest pannanum. Oru **Small Cap** fund **50%** return kuduththirukkum. Aana adhu **next year** **30% down** aaga kooda chance irukku. Nee unga **risk appetite**-kku eththa **fund**-a dhaan select pannanum.

IV. ğŸ›‘ Mistake 3: Trying to **Time the Market**

**"Market innum konjam keezhe varattum, aprom Lumpsum-a podalaam!"** nu yosikradhu dhaan indha mistake.

Time in the Market is Key

**Market** eppa **low** aagum nu **yaarukkum** theriyadhu. **Wait** panradhunaala, nee **SIP**-la kidaikkum **long-term compounding**-a **miss** pannuvai. **SIP** dhaan **market timing**-kku **kavalai** padaama **consistent growth**-kku help pannum.

V. ğŸ›‘ Mistake 4: Using **Debt Funds** for **Long-Term Goals**

**Retirement** (15+ years) maadhiri **long-term goals**-kku **Debt Fund** use panradhu **romba risk** da.

The Inflation Trap

**Debt Fund** **safety** kudukkum, aana **return** **inflation**-a vida **romba kammi** dhaan irukkum. **15 years** aprom, unga money value **romba kammi** aagidum. **Long-term goals**-kku **Equity Funds** (Large/Mid Cap) dhaan use pannanum.

VI. ğŸ›‘ Mistake 5: **Panic** Panni **Sell** Panradhu

**Market crash** aagum bodhu, **bhayaththula** **SIP**-a **stop** panradhu illa **units**-a **sell** panradhu dhaan **most dangerous mistake**.

The MFD's Role

Nee **loss**-la sell panninaa, adhu dhaan **real loss**. **Market** **marubadiyum recover** aagum. Andha **tough time**-la, **Vignesh** maadhiri oru **MFD** dhaan ungalukku **advice** kuduththu, **emotional decision** edukkaama **patience**-a maintain panna help pannuvanga.
