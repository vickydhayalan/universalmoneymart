2. Sharpe Ratio — ⭐ Gold Standard

Sharpe Ratio = (Fund Return − Risk-Free Rate) ÷ Standard Deviation

Risk-Free Rate = India 10-yr Govt Bond (~7%)

Example: Fund 15%, Risk-free 7%, Std Dev 12% → Sharpe = (15-7)÷12 = 0.67

Sharpe ValueInterpretation
< 0.5Poor — Risk not worth it
0.5 – 1.0Acceptable
1.0 – 1.5Good ✓
1.5+Excellent! ✓✓
⚠️ Important

Same category funds-ல மட்டும் compare பண்ணு! Small Cap Sharpe vs Large Cap Sharpe = invalid comparison.

3. Alpha — Fund Manager Skill

Alpha = Fund Return − Benchmark Return
ScenarioAlphaVerdict
Fund 16%, Nifty 50 12%+4%Manager adds value!
Fund 10%, Nifty 50 12%-2%Index fund better!
Fund 12%, Nifty 50 12%0%Go with Index!
✅ Key

Consistent positive alpha for 3-5 years = real manager skill. One year positive = could be luck!

4. Beta — Market Sensitivity

Beta = How fund moves relative to benchmark. Benchmark Beta = 1.0

1.0Moves with marketIndex fund investors
1.0–1.3Slightly more volatileModerate risk

5. Maximum Drawdown

Peak-ல இருந்து trough-வரை maximum fall. ₹100 → ₹180 → ₹108 = -40% Max Drawdown.

6. Rolling Returns — Consistency Check

Every possible 3-year period-ல returns check. 70%+ periods beat benchmark = consistent performer!

📊 Why This Matters

5/6 periods beat = Consistent. 2/6 periods beat = Lucky in 2 years, not skilled.
Rolling returns tell truth that point-to-point returns hide!

8. Practical Checklist

  • ⭐ Sharpe Ratio: 1.0+ good, 1.5+ excellent. Same category only
  • α Alpha: Positive consistently (3-5 years)
  • β Beta: Match your risk preference
  • 📉 Max Drawdown: Below category average
  • 📈 Rolling Returns: 70%+ periods beat benchmark
